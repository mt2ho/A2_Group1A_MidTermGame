Group 1A

## Description

Planet Amtcha is a café simulation game inspired by Wernicke’s aphasia, a disorder that affects language comprehension and producing language. Players run a café on an alien planet where customers give confusing orders. The player must read, decipher the words, and correctly input the combinat orders into a POS system. Success depends on maintaining customer satisfaction by entering accurate orders before time runs out Wrong orders, out of time, and asking for clarification are factors that can decrease the meter.

## Setup and Interaction Instructions:

Setup:

- players can click on instructions in the splashsreen before starting
- Alien customers arrive with written orders that may appear confusing or difficult to understand.
- A POS system is available for the player to input drink combinations
- Each customer has a satisfaction meter that reflects how well their order is handled

Interactions:

- Read the alien customer’s order carefully.
- Decipher and memorize the requested items or combinations.
- after pressing okay, you go into POS system and click the icons to match the order combinations, and submit before the time runs out
- To successfully complete the level/day, maintain the satisfaction meter above the required threshold

Submit the order to serve the customer.

## Iteration Notes (Post-Playtest)

During playtesting we observed both positive engagement and areas for improvement:

Gameplay Functionality

- Some button interactions (particularly the POS system) were inconsistent and did not always register clicks.
- The “Huh?” clarification mechanic occasionally triggered alien movement inconsistently.
- Only one drink variation (strawberry matcha) currently appears, which limited gameplay variety.

Visual Feedback

- Playtesters strongly enjoyed the visual design, describing the art style as “cute,” “adorable,” and “bouncy.”
- The floating alien animation and font choice were praised for matching the cozy aesthetic.

Player Interpretation & Challenge

- Some alien words were easy to interpret (e.g., “hot”), while others were more confusing (e.g., “iced”).
- Players were engaged by the decoding mechanic but suggested that difficulty should increase gradually.

Gameplay Ideas from Testers

- Add a timer system for drink orders to increase pacing and challenge.
- Introduce more drink variations and toppings over time.
- Potentially allow multiple customers with separate orders to create prioritization decisions.

Based on playtest feedback, the core changes that we will make for A2 is introducing more flavours and toppings for different levels/ randomization of drinks, a timer + progression mechanics, and expanding the the alien language system to create more challenge and variation.

## Iteration Notes (Post-Showcase)

- The first improvement we will make is making sure to introduce to the player what the options are (toppings, flavours, etc..) while the alien customer is saying their order when it is a new day and level. We could possibly do this by adding the options/list onto the POS system.

- The next improvement is just adding more feedback so customers can know what options need to be clicked, and pressing "huh" would make the satisfaction meter go down.
